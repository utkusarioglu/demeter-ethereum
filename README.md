# Demeter - Ethereum

Ethereum smart contract repo for the demeter project

## Tech Stack

| Component       | Tool used             |
| --------------- | --------------------- |
| Framework       | Hardhat               |
| Deployer        | Hardhat Deploy Plugin |
| Test framework  | Mocha                 |
| Assertion tools | Chai, Waffle          |
| Type generator  | typechain             |
| Web3 surface    | ethersjs              |
| Formatting      | Prettier              |
